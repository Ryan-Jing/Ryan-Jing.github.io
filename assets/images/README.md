# Images Folder

## Add Your Images Here

Place your general images (profile photo, etc.) in this folder.

### Recommended Images:
1. **Profile Photo**
   - File: `profile-photo.jpg` or `profile-photo.png`
   - Size: 500x500 pixels (square)
   - Use in: About section

### Image Optimization Tips:
- Keep file sizes under 500KB
- Use JPG for photos
- Use PNG for graphics/icons
- Compress images before uploading
- Use descriptive filenames (no spaces)

### Tools for Compression:
- TinyPNG: https://tinypng.com
- Squoosh: https://squoosh.app
- ImageOptim (Mac): https://imageoptim.com
