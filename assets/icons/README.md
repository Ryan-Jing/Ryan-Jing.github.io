# Icons Folder

## Add Your Favicon Here

Place your website icon as `favicon.png` in this folder.

### Favicon Requirements:
- **Format:** PNG (or ICO)
- **Size:** At least 192x192 pixels (512x512 recommended)
- **Name:** `favicon.png`

### Source Location:
Your icon can be found at:
```
/Users/ryanjing/Library/Mobile Documents/com~apple~CloudDocs/Ryan/Uni/Uni Files/Portfolio Stuff
```

### How to Add:
1. Copy your PNG file from the source location
2. Rename it to `favicon.png`
3. Paste it in this folder (`assets/icons/`)

The website will automatically use it as the tab icon!
